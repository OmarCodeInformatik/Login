# 🔐 Login for Websites

Ein modernes und interaktives Login-System für Webseiten, entwickelt mit reinem Vanilla JavaScript. Das Projekt enthält Funktionen wie Registrierung, Passwort-Zurücksetzen und eine benutzerfreundliche Oberfläche mit Animationen.

## ✨ Technologien

- HTML  
- CSS  
- JavaScript (Vanilla)

## 🚀 Features

- Login- und Registrierungs-System  
- „Passwort vergessen“-Funktion  
- „Remember Me“-Option  
- Pflichtfelder (alle Eingaben müssen ausgefüllt werden)  
- Modernes und sauberes UI-Design  
- Hover-Animationen für bessere Interaktion  
- Weiterleitung nach erfolgreichem Login auf eine Bestätigungsseite  
- Fehlermeldungen bei unvollständigen Eingaben  

## 📍 Der Entwicklungsprozess

Das Ziel dieses Projekts war es, ein funktionales und gleichzeitig modernes Login-System ohne Frameworks zu erstellen. Alles wurde von Grund auf mit HTML, CSS und JavaScript entwickelt.

Der Fokus lag auf einem sauberen Design und einer guten Benutzererfahrung. Durch Hover-Animationen und visuelle Effekte wirkt die Oberfläche lebendig und interaktiv.

Zusätzlich wurde sichergestellt, dass alle Eingabefelder ausgefüllt werden müssen, bevor der Benutzer sich einloggen oder registrieren kann. Nach einem erfolgreichen Login wird der Nutzer auf eine Seite weitergeleitet, auf der eine Bestätigung angezeigt wird.

## 🚦 Projekt starten

1. Repository klonen oder herunterladen  
2. Projektordner öffnen  
3. `index.html` im Browser öffnen
